##Run Locally

Install all the dependencies:

    npm install (you may need to prefix this with sudo if you're on Mac)

Run the app:

    node server.js

Then navigate to `http://localhost:3000`



    npm install -g nodemon

Then run the following instead of `node server.js`:

    nodemon server.js
